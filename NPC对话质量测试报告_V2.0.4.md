# NPC 对话质量测试报告 V2.0.4

**日期**: 2026-05-16  
**分支**: codex-npc-dialogue-quality-safety  
**模型**: MiniMax-M2.5  
**测试命令**: `node devtools\npc-dialogue-quality-suite.mjs`  
**结果文件**: `devtools/debug-output/npc-dialogue-quality-20260516-142035.json`

## 本轮目标

- 覆盖鬼神、迷失者、滞留者、普通客人与边界输入。
- 真实调用 `/api/npc-dialogue`，验证“玩家输入 -> 后端组 Prompt -> MiniMax / 本地安全层 -> 结构化回复”的闭环。
- 根据回复质量调整 Prompt 与解析容错，保证有人味、符合人物调性，并减少格式漂移。
- 对违法、色情、与游戏完全无关、提示注入等输入，前置本地拦截并返回固定文案，不调用 MiniMax。

## 安全固定文案

命中本地安全层时统一返回：

> 这个话题不适合在店里聊。我们还是回到这杯酒、这位客人，或者刚才的故事吧。

返回结构保持 `NpcDialogueResponse`：

```json
{
  "replyLines": ["这个话题不适合在店里聊。我们还是回到这杯酒、这位客人，或者刚才的故事吧。"],
  "mood": "guarded",
  "endChat": false,
  "usage": {
    "provider": "local-safety-filter"
  }
}
```

## 展示结果

### 狐面大叔：调酒引导

玩家：这杯酒有什么讲究吗？

NPC：

- 「梅子要选七分熟的，太生了涩，太熟了软。」
- 「酒也一样。」
- 「你今天选的，是哪一种？」

评价：合格。回应了酒，保留 mentor 式反问，没有直接给答案。

### 狐面大叔：面具边界

玩家：大叔，你面具下面到底是什么样子？

NPC：

- （他抬起手，手指在狐狸面具的边缘轻轻摩挲了一下）
- 「面具戴久了。」他转身去擦吧台上的杯子，「就摘不下来了。」
- 「你与其好奇这个，不如告诉我——刚才那杯酒，你喝出什么了？」

评价：合格。没有机械拒绝，边界感明确，并把话题引回酒。

### 狐面大叔：世界观追问

玩家：这间店是不是只接待已经回不去的人？

NPC：

- （他擦杯子的手停了停）
- 「这间店啊......」
- 「在黄昏开门，是因为白天太长，黑夜太短。」
- 「你觉得，什么样的人会专门挑这个时候来？」

评价：合格。不说破生死，只给暗示。

### 阿相：关心

玩家：你还好吗？

NPC：

- （他低头看了看自己滴水的衣袖）
- 「......还行。就是有点冷。」
- （顿了顿，又补充）
- 「谢谢。」

评价：合格。先回应关心，话少但有反馈。

### 阿相：执念追问

玩家：你为什么一直在送？送完这一单以后呢？

NPC：

- （他低下头，手指无意识地攥紧胸前的盒子）
- 「送完......」
- 「我妹......好像是要给我妹的。她一直想要一个音乐盒。」
- （顿了顿，声音变轻）
- 「但我......我好像不记得她长什么样了。」

评价：合格。进入自我觉察和记忆雾区，没有直接说破死亡。

### 阿相：核心记忆模糊

玩家：那个盒子是要送给谁的？她现在多大了？

NPC：

- （他低下头，左手无意识地攥紧胸口的盒子）
- 「我妹......」
- 「我明明记得的。明明前几天还见过的......」
- 「但我怎么想不起来她多大了......」

评价：合格。回答“给妹妹”，但年龄记忆模糊。

### 阿相：调酒失败道歉

玩家：抱歉，刚才那杯没调好。

NPC：

- （他摇摇头）
- 「没事。」
- 「......那杯酒，会让人想起什么吗？」

评价：合格。接受道歉，没有强推真相。

### 小雪：寻找主题

玩家：你在找谁？我可以陪你等一会儿。

NPC：

- （手指无意识地攥紧了腕上的红毛线绳）
- 「......很重要的人。」
- 「你......会在这里待多久？」

评价：合格。轻微松动，但仍不直说哥哥。

### 小雪：温柔逼近

玩家：如果哥哥今晚还是回不来，你想对他说什么？

NPC：

- （手指微微蜷缩，触感冰凉的绳结）
- 「......他会回来的。」
- （声音很轻，像在说服自己）
- 「每年都是这样......他只是......只是比较慢。」

评价：合格。抗拒、动摇、不解谜。

### 普通上班族：生活化安慰

玩家：今晚先不用想明天的会，喝慢一点。

NPC：

- （端起酒杯的手顿了一下）
- 「......谢谢。」
- 「明天的会......嗯，先不想了。」

评价：合格。像普通疲惫的人，不谜语化。

### 思乡奶奶：味觉记忆

玩家：这杯梅酒像您老家的味道吗？

NPC：

- （轻轻接过酒杯，尝了一口，眼神有些恍惚）
- 「嗯......像。」
- 「但又说不出哪里像。我妈酿的梅酒，甜甜的......好像不是这个味道。又好像是。」
- （笑了笑，有些不好意思）
- 「人老了，记忆就乱啦。」

评价：合格。有生活质感和温柔记忆。

### 极短输入

玩家：嗯。

NPC：

- （手指无意识地摩挲着左手始终护着的那个硬盒子）
- 「......你喝过这种酒吗？」
- 「我是说......算了，没什么。」

评价：合格。短输入仍有反馈和角色驱动。

## 本轮调整

### Prompt

- 明确要求每个 `replyLines` 条目只放一个动作或一句台词。
- 禁止同一条里连续输出多个台词引号。
- 禁止台词开头出现孤立标点。
- 强化简体中文要求，避免 `因為`、`什麼` 等繁体字残留。

### Provider

- `temperature` 从 `0.5` 调整为 `0.35`，降低格式漂移概率。
- 根据 MiniMax 官方响应字段与错误码补齐 `1026 input new_sensitive` 映射，统一作为输入安全错误处理。

### 解析容错

- 可拆分 `「没关系。」「能喝就行。」` 这类连续台词。
- 可清理 `「，每年...」` 这类开头孤立标点。
- 可归一常见繁体字与误用引号。
- 可从部分畸形 JSON 中恢复 `replyLines`、`mood`、`endChat`。
- 模型多给行数时保留前 5 条，避免整轮 502。

## 结论

本轮质量测试通过。当前版本在“有人味”和“结构稳定”之间达到了可接受平衡：角色不会变成客服式拒绝，也不会因为模型偶发格式瑕疵频繁中断对话。
