# 酒谱总览

---

## 日式经典系列

### 纯米嗨棒
- **配方**: 纯米清酒 + 苏打水
- **情感**: 怀旧、希望
- **描述**: 纯米清酒与苏打水的简单结合，日本人的日常。

### 甘酒嗨棒
- **配方**: 甘酒 + 苏打水
- **情感**: 留恋、希望
- **描述**: 甘酒与苏打水的相遇，温暖的甜蜜，像小时候的味道。

### 抹茶椰林
- **配方**: 抹茶 + 椰浆 + 蜂蜜
- **情感**: 迷茫、慰藉
- **描述**: 抹茶与椰浆的相遇，蜂蜜的甜是南国的温柔，茶香与椰风的私语。

### 梅酒嗨棒
- **配方**: 纯米清酒 + 樱桃利口酒 + 苏打水
- **情感**: 怀旧、留恋、希望
- **描述**: 清酒与梅酒的结合，日本的夏天，梅子的酸甜。

### 柚子蜂蜜
- **配方**: 纯米清酒 + 柚子酱 + 蜂蜜
- **情感**: 怀旧、共鸣、慰藉
- **描述**: 清酒与柚子酱、蜂蜜的温柔组合，柚子的清香，蜂蜜的温润。

### 蜜柚茶
- **配方**: 纯米清酒 + 乌龙茶 + 蜂蜜
- **情感**: 怀旧、迷茫、慰藉
- **描述**: 清酒与乌龙茶、蜂蜜的结合，是成熟果实的温柔，秋日的私语。

### 烧酌嗨棒
- **配方**: 烧酌 + 苏打水
- **情感**: 执念、希望
- **描述**: 烧酌与苏打水的简单结合，九州的夜晚，芋香与气泡的邂逅。

### 芋感HIGHBALL
- **配方**: 烧酌 + 姜汁汽水 + 柠檬汁
- **情感**: 执念、希望
- **描述**: 烧酌与姜汁汽水、青柠的碰撞，汗水与笑声，辛辣中的清醒。

### 泡盛嗨棒
- **配方**: 泡盛 + 苏打水
- **情感**: 怀旧、希望
- **描述**: 泡盛与苏打水的相遇，琉球的海风，南海的风带着稻香。

### 泡盛荔枝
- **配方**: 泡盛 + 橙汁 + 樱桃利口酒
- **情感**: 怀旧、重逢、留恋
- **描述**: 泡盛与橙汁、荔枝香的结合，是海岛的风情，热带的风土。

## 经典系列

### 血腥玛丽
- **配方**: 伏特加 + 番茄汁 + 苦精 + 柠檬 + 黑胡椒 + 盐
- **情感**: 忘却、离别、孤独、迷茫、执念
- **描述**: 红的像血，辣得像生活，每一口都是真实的清醒。

### 教父
- **配方**: 威士忌 + 杏仁酒
- **情感**: 孤独、宽恕
- **描述**: 威士忌与杏仁的邂逅。醇厚中带着一丝苦涩，像是成年人的沉默。

### 玛格丽特
- **配方**: 龙舌兰 + 君度橙酒 + 柠檬汁 + 盐
- **情感**: 执念、留恋、希望、离别
- **描述**: 龙舌兰的烈、君度的甜、柠檬的酸——沙漠里的绿洲。

### 马丁尼
- **配方**: 伦敦干金酒 + 苦艾酒
- **情感**: 迷茫、孤独
- **描述**: 金酒与苦艾酒的相遇，简洁而深邃。鸡尾酒之王。

### 曼哈顿
- **配方**: 威士忌 + 苦艾酒 + 樱桃利口酒
- **情感**: 孤独、留恋
- **描述**: 威士忌与甜苦艾酒的经典组合，纽约的夜色。

### 古典
- **配方**: 威士忌 + 方糖 + 黑胡椒
- **情感**: 孤独、留恋、执念
- **描述**: 威士忌、糖、苦精——回到最初的鸡尾酒，最简单的往往最经典。

### 螺丝刀
- **配方**: 伏特加 + 橙汁
- **情感**: 忘却、重逢
- **描述**: 伏特加与橙汁的邂逅，苏联工人的智慧，清晨的太阳。

### 伏特加汤力
- **配方**: 伏特加 + 通宁水
- **情感**: 忘却、孤独
- **描述**: 伏特加与通宁水的结合，简洁的刺激，透明的清醒。

### 金菲士
- **配方**: 伦敦干金酒 + 柠檬汁 + 薄荷
- **情感**: 迷茫、希望、平静
- **描述**: 金酒、柠檬、薄荷的清爽组合，气泡在舌尖跳舞。

### 莫吉托
- **配方**: 陈年朗姆酒 + 柠檬汁 + 薄荷 + 海盐
- **情感**: 重逢、希望、平静、离别
- **描述**: 朗姆酒与薄荷的清爽相遇，青柠的酸涩是海岸线的味道。

### 自由古巴
- **配方**: 陈年朗姆酒 + 可乐 + 柠檬汁
- **情感**: 重逢、忘却、希望
- **描述**: 朗姆酒与可乐的碰撞，是雪茄与蔗糖的南国。

## 传奇系列

### 蓝色夏威夷
- **配方**: 伏特加 + 橙汁 + 椰浆 + 蓝柑桂
- **情感**: 忘却、重逢、慰藉、遗忘
- **描述**: 伏特加、蓝柑桂、椰浆——度假的颜色，蓝天白云下的梦境。

### 新加坡司令
- **配方**: 伦敦干金酒 + 橙汁 + 柠檬汁 + 樱桃利口酒 + 方糖
- **情感**: 迷茫、重逢、希望、留恋
- **描述**: 金酒与柠檬、樱桃的相遇，殖民地的记忆与东方的优雅。

### 僵尸
- **配方**: 伏特加 + 龙舌兰 + 橙汁 + 柠檬汁 + 樱桃利口酒
- **情感**: 忘却、执念、重逢、希望、留恋
- **描述**: 三种酒的热带组合，危险的美丽，喝完你会知道为什么叫僵尸。

### 飞行
- **配方**: 伦敦干金酒 + 柠檬汁 + 樱桃利口酒 + 接骨木花露
- **情感**: 迷茫、希望、留恋、共鸣
- **描述**: 金酒、柠檬、樱桃、接骨木——云端的滋味，起飞前最后一杯。

### 边车
- **配方**: 龙舌兰 + 柠檬汁 + 君度橙酒
- **情感**: 执念、希望、留恋
- **描述**: 龙舌兰、君度、柠檬——摩托车旅行的味道，公路的浪漫。

### 净化之泪
- **配方**: 生命之水 + 通宁水 + 海盐
- **情感**: 释然、孤独、离别
- **描述**: 生命之水与通宁水的结合，海盐带来远方的咸涩，泪水洗净铅华。

### 极光
- **配方**: 生命之水 + 抹茶 + 晨曦微光
- **情感**: 释然、迷茫、希望
- **描述**: 生命之水与抹茶的相遇，晨曦微光带来重生的温暖，黎明前的黑暗。
